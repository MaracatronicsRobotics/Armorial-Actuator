# Armorial-Actuator
This repository contains the actuator software used by Maracatronics Robotics Team and Research Lab at Small Size League to communicate with the robots.

## How to use

```
sudo sh protobuf.sh

mkdir build
cd build
qmake ..
make

./Armorial-Actuator
```
